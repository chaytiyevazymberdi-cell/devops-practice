# DevOps Practice

Практические задания по DevOps которые я выполнил самостоятельно.

## Что изучил
- Linux — файловая система, права доступа, процессы
- Bash скрипты — автоматизация задач
- SSH — настройка доступа по ключу
- Nginx — веб-сервер, конфигурация, деплой
- Docker — контейнеры, Dockerfile, образы
- Docker Compose — управление несколькими контейнерами
- Git — контроль версий
- CI/CD — GitHub Actions pipeline

## Технологии
Linux | Bash | Nginx | Docker | Git | GitHub Actions
